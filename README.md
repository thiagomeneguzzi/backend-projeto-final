## YourPet - Backend (NestJS)

### Setup
```
npm install
```

### Compiles for development (hot-reloads)
```
npm run start:dev
```

### Compiles and minifies for production
```
npm run build
```

### Commits
```
'type'(Jira tag issue): 'message'
```
